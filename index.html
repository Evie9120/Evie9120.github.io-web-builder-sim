🎮 How to Play:

  1. 👤 A client will ask you to build a website with specific blocks.
  2. 🧱 Drag blocks from the left sidebar into the Construction Zone below.
  3. 📦 You can put blocks inside containers (like Box, Card, Grid).
  4. ✏️ Click the ✏️ button on a block to edit its content.
  5. 👾 Drop unwanted blocks onto the Trash Monster to delete them.
  6. 📜 Press Show Code to see the real HTML you've built!
  7. ✅ When ready, click Send to Client to earn money!
  8. 🏆 Complete all 20 jobs to win!
